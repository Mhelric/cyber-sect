Per the **Official Game Rules**, every Operator begins a run with a **15-Card Deck**.

* **10 Core Cards:** These are the specialized, immutable tools unique to that Job Class.
* **5 Flex Slots (Standard Issue):** These slots are initially filled with "Basic Protocols." As the player acquires new cards from the Shop or Data Logs, they will swap these out to customize their build.
